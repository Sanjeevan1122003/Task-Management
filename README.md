# Task Management Application

A role-based task management application built with React, featuring drag-and-drop functionality and localStorage persistence.

## Features

- **Role-based Access Control**:
  - Admin: Create, assign, reassign, and delete tasks
  - User: View assigned tasks and mark them as completed
- **Drag-and-Drop**: Admin can reassign tasks by dragging them onto users
- **LocalStorage Persistence**: All data persists across browser sessions
- **Responsive Design**: Works on desktop and mobile devices
